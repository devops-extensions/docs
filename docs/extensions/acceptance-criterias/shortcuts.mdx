# Keyboard Shortcuts

## Criteria Panel - Checklist & Scenario

While you have one of the text fields focused, you can use the following shortcuts

| Shortcut                             | Description                                                                                           |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------- |
| <kbd>Ctrl</kbd>+<kbd>Enter</kbd>     | `Checklist`: Add a new checklist item. `Scenario`: Move the focus to the toolbar to select a new step |
| <kbd>Ctrl</kbd>+<kbd>ArrowUp</kbd>   | Move item up                                                                                          |
| <kbd>Ctrl</kbd>+<kbd>ArrowDown</kbd> | Move item down                                                                                        |
| <kbd>Ctrl</kbd>+<kbd>Delete</kbd>    | Remove item                                                                                           |

## Browser default

| Shortcut                        | Description                    |
| ------------------------------- | ------------------------------ |
| <kbd>Tab</kbd>                  | Move focus to next element     |
| <kbd>Shift</kbd>+<kbd>Tab</kbd> | Move focus to previous element |
